# Contributing

Thank you for your interest in contributing to this project.

## Author
Floyd Steev Santhmayer

## How to contribute
- Fork the repository.
- Create a feature branch.
- Add tests for any new functionality.
- Make sure `pre-commit` hooks pass locally before opening a PR.
- Open a pull request and describe the change clearly.

## Code of Conduct
Be professional and respectful. This project follows a standard code of conduct.
